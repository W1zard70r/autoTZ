# autoTZ — Генератор технических заданий на базе LLM

LLM-пайплайн для автоматической генерации технических заданий из переписок команды и других источников.

## Архитектура

```
schemas/            — Pydantic-модели данных
  templates/        — Строгие шаблоны ТЗ (ГОСТ, бытовой, IT, строительный, инженерный)
  graph.py          — Модели графа знаний
  document.py       — Модели документов и источников
  enums.py          — Перечисления (типы шаблонов, секций, узлов)

core/               — Ядро пайплайна
  pipeline.py       — Оркестратор (шаги: извлечение → слияние → компиляция)
  miner.py          — Layer 1: извлечение знаний из источников
  windowing.py      — Семантическое окно для разбивки чатов
  merger.py         — Layer 2: слияние графов, дедупликация, конфликты
  compiler.py       — Layer 3: заполнение шаблона ТЗ по графу
  translator.py     — Перевод готового документа между языками

interfaces/         — Интерфейсы ввода
  base.py           — Абстрактный интерфейс
  cli.py            — Консольный интерфейс (Human-in-the-Loop)
  api.py            — API-адаптер для подключения к бэкенду
  test_runner.py    — Автоматическая прогонка на тестах

utils/              — Утилиты (LLM-клиент, эмбеддинги, логирование)
main.py             — Точка входа
```

## Шаблоны ТЗ

Каждый шаблон — строгая Pydantic-схема с валидацией заполненности и рендерингом в Markdown.

## Запуск

```bash
python main.py cli   # Интерактивный режим
python main.py test  # Автоматические тесты
```

## Локализация

Генерация текста ТЗ — сразу на русском языке через системные промпты компилятора.
Модуль `core/translator.py` доступен для перевода готового документа при необходимости.
